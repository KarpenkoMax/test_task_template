# Часть 1 (15–20 мин) — Pixel JS (без LLM)

Цель: сделать минимальный website pixel, который надёжно доставляет события и не мешает работе страницы.

Где работать:
- `frontend/src/pixel/pixel.ts`
- тесты: `frontend/src/pixel/pixel.test.ts`

Запуск тестов:
```bash
cd frontend
npm test
```

Что важно:
- Подумайте о надёжности доставки при временных сбоях сети.
- Учтите выгрузку страницы (page unload) и влияние на производительность.
- Дизайн очереди/ретраев и формат хранения — на ваше усмотрение.

Важно качество решения и объяснение компромиссов.
